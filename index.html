Mes Outils

Docker

Docker est une plateforme de virtualisation légère qui permet de créer, déployer et exécuter des applications de manière isolée. Elle simplifie le déploiement et la gestion des applications, favorise la portabilité et facilite la collaboration entre les équipes de développement et d'exploitation.

SQL

SQL est un langage de programmation pour les bases de données relationnelles, utilisé pour récupérer, insérer, mettre à jour et supprimer des données de manière efficace. C'est un outil essentiel pour la gestion des données

Java

Java est un langage de programmation polyvalent, orienté objet et portable, utilisé pour développer des applications de bureau, Web et mobiles. Il est apprécié pour sa syntaxe claire, sa bibliothèque standard étendue et sa communauté de développeurs active.

Python

Python est un langage de programmation polyvalent et convivial, connu pour sa syntaxe claire et concise. Il est largement utilisé dans le développement d'applications, l'analyse de données et l'apprentissage automatique en raison de sa flexibilité et de sa productivité élevée.

HTML

HTML (Hypertext Markup Language) est le langage de balisage standard pour la création de pages Web. Il utilise des balises pour structurer le contenu et fonctionne avec CSS et JavaScript pour la présentation et l'interactivité. HTML est simple à apprendre et compatible avec les navigateurs modernes.

CSS

CSS est un langage de style utilisé pour contrôler la présentation des pages Web. Il permet de définir les couleurs, les polices, les marges et les dispositions des éléments HTML, contribuant ainsi à créer des interfaces Web attrayantes.

JavaScript

JavaScript est un langage de programmation polyvalent utilisé pour ajouter des fonctionnalités interactives et dynamiques aux pages Web. Il est exécuté côté client, offre de nombreuses possibilités et dispose d'une vaste communauté de développeurs.

Figma

Figma est un outil de conception d'interfaces utilisateur basé sur le cloud, utilisé pour créer des maquettes et des prototypes interactifs. Il offre une interface conviviale et des fonctionnalités avancées
